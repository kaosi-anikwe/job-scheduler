# Frontend

> React-based dashboard UI for the job scheduler.
> This directory will contain the frontend application.

## Planned Views

- **Dashboard** — job counts by status
- **Jobs Table** — ID, type, priority, status, retry count, scheduled time, interval, created time
- **Create Job Form** — all fields, self-explanatory to use
- **DLQ View** — failed jobs with error details and manual retry button
